# scripts/

Utilidades: seeders, generadores de data mock, helpers de setup.

## Qué va aquí
| Script | Propósito |
|---|---|
| `seed.*` | Cargar [`../datasets/`](../datasets/) en la base de datos |
| `generate-leads.*` | Generar leads mock realistas por canal |
| `reset.*` | Limpiar y re-sembrar para la demo |

## Regla de oro para el hackathon
Un solo comando debe dejar la demo lista: **seed + run**. Si el jurado (o ustedes a las 3am) necesita más de un paso para arrancar, algo está mal.
