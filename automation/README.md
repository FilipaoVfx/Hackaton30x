# automation/

Workflows de automatización (ej. **n8n**) que conectan canales y disparan el flujo sin intervención humana.

## Responsabilidad
- Recibir webhooks de los canales (Meta Lead Ads, WhatsApp, formularios web).
- Disparar la ingesta hacia el backend.
- Orquestar el **flujo de nutrición** (recordatorios, reactivación de leads COLD).

## Qué guardar aquí
- Exports de workflows de n8n (`*.json`).
- Diagramas de los flujos automatizados.
- Documentación de cada webhook y su trigger.

## Por qué importa para el jurado
Demuestra que el flujo **corre solo**, cumpliendo el criterio: _"el flujo completo se puede recorrer sin intervención del equipo"_.
